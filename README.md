# 준석's dev_training

학원에서 배운 내용을 복습하고, 실습 문제를 풀며  
프로젝트를 통해 개발 역량을 키우기 위한 개인 학습 저장소입니다.

---

## 📌 목적
- 수업 내용 복습 및 정리
- 실습을 통한 개념 이해
- 프로젝트 경험 축적
- 오류 기록을 통한 실수 반복 방지

---

## 📂 폴더 구조

```text
dev_training/
 ┣ notes/         # 수업 내용, 개념 정리
 ┣ practice/      # 실습 문제 풀이
 ┣ projects/      # 개인 / 팀 프로젝트
 ┣ error-notes/   # 오답노트 및 에러 기록
 ┣ setup/         # 개발 환경 설정
 ┣ snippets/      # 자주 쓰는 코드 조각
 ┣ resources/     # 참고 자료 모음
 ┗ retrospective/ # 회고 및 정리
